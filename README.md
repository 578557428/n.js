n.js
====

> 轻量级的工具集合（兼容傻x ie6）

##方法支持##

* DOM 选择器 n()
* 事件绑定 n.on()
* 解除事件绑定 n.un()
* 触发对象的指定事件 n.fire()
* 创建dom元素 n.md()
* 数组/普通对象/字符串/类数组遍历 n.forEach()
* 格式化日期 n.dFormat()
* 去除字符串的前后空格 n.trim()
* url参数解析 n.queryUrl()
